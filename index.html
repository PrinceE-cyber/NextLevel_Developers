🎯 Objective

By the end of this lesson you will understand what HTML is, why it exists, and how to create your very first web page with just a text editor.

📖 Explanation

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. Think of HTML as the skeleton of a web page — it gives everything its shape and meaning.

HTML is made up of elements. Each element is written using tags. Most tags come in pairs: an opening tag <tag> and a closing tag </tag>, with the content in between.

Every HTML document starts with a standard boilerplate. The browser reads this top-to-bottom to know what to display.

💻 Code Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is my first web page.</p>
</body>
</html>

Save this as index.html and open it in your browser. You should see "Hello, World!" as a big heading!

✏️ Practice Task

  1. Create a new file called index.html on your computer.
  2. Type (don't copy-paste!) the boilerplate code above.
  3. Change the <title> to your own name.
  4. Change the <h1> text to a personal greeting.
  5. Add a second <p> tag below the first with one sentence about yourself.
  6. Open the file in a browser — congratulations, you built a web page! 🎉

🎯 Objective

Learn how to format text using HTML heading tags, paragraphs, bold, italic, and how to create ordered and unordered lists.

📖 Explanation

HTML provides six heading levels: <h1> through <h6>. <h1> is the most important (biggest) and should appear only once per page. Use headings to create a clear hierarchy of content.

To emphasise text, use <strong> for bold importance and <em> for italic emphasis. These are semantic tags — they tell the browser (and screen readers) that the content has special meaning, not just visual styling.

Lists come in two flavours: unordered (<ul>) for bullet points and ordered (<ol>) for numbered steps. Each item inside either type uses an <li> tag.

💻 Code Example

<!-- Headings -->
<h1>Main Title</h1>
<h2>Sub-heading</h2>
<h3>Smaller sub-heading</h3>

<!-- Emphasis -->
<p>This word is <strong>bold</strong>.</p>
<p>This word is <em>italic</em>.</p>

<!-- Unordered list (bullet points) -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Ordered list (numbered) -->
<ol>
  <li>Open your text editor</li>
  <li>Write your HTML</li>
  <li>Save and refresh the browser</li>
</ol>

✏️ Practice Task

  1. Open your index.html from Day 1 inside the <body>.
  2. Add an <h2> that says "My Favourite Foods".
  3. Below it, create a <ul> with at least 3 food items.
  4. Add an <h2> that says "Steps to Make Tea".
  5. Create an <ol> with the numbered steps.
  6. Use <strong> on one important word in a paragraph.

🎯 Objective

Understand what CSS is and how it connects to HTML. Learn to change text colour, background colour, and font properties using CSS rules.

📖 Explanation

CSS stands for Cascading Style Sheets. While HTML provides the structure, CSS provides the style — colours, fonts, spacing, layout, and more.

A CSS rule has two parts: a selector (which HTML element to target) and a declaration block (the styles to apply). Each declaration is a property:value pair, separated by a semicolon.

The best practice is to write CSS in a separate file (e.g. style.css) and link it to your HTML using a <link> tag inside <head>. This keeps your code organised and lets you style multiple pages from one file.

💻 Code Example

/* Link CSS in your HTML <head>: */
<!-- <link rel="stylesheet" href="style.css" /> -->

/* ── style.css ── */

/* Target the body element */
body {
  background-color: #fdf6ec;
  color: #2c2c2c;
  font-family: Georgia, serif;
  font-size: 18px;
}

/* Target all h1 elements */
h1 {
  color: #c0392b;  /* a rich red */
  font-size: 2.5rem;
}

/* Target paragraphs */
p {
  line-height: 1.7;
  margin-bottom: 1rem;
}

✏️ Practice Task

  1. Create a new file called style.css in the same folder as index.html.
  2. Add the <link> tag inside your HTML <head>.
  3. Set a background colour on body using any hex colour you like (try coolors.co for ideas).
  4. Change the h1 colour to something bold.
  5. Set the font-family on body to Arial, sans-serif.
  6. Increase the font-size of paragraphs to 1.1rem.
