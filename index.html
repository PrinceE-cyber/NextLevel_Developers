What is HTML?
The Simple Definition

HTML = HyperText Markup Language. It is the skeleton of every webpage on the internet. It tells the browser: "here is a heading, here is a paragraph, here is an image." Every website you have ever visited — Google, YouTube, Instagram — is built on HTML at its core.

💡 Think of it this way
If a webpage were a house — HTML is the walls, floors, and rooms. CSS is the paint and furniture. JavaScript is the electricity and plumbing. Today we build the house structure.
How HTML Works

HTML uses tags to label content. Tags come in pairs — an opening tag and a closing tag. The browser reads these tags and knows how to display the content.

example
<h1>This is a heading</h1>
<p>This is a paragraph of text.</p>
<a href="https://google.com">Click me</a>

Opening tag: <h1>  ·  Closing tag: </h1>  ·  The content goes in between.

02
Tags You Must Learn Today
The Document Structure Tags

Every single HTML file you ever write must start with these — no exceptions.

<!DOCTYPE html>Tells the browser: this is HTML5. Always first line.
<html>Wraps the entire page. Everything lives inside this.
<head>Hidden info: page title, links to CSS. Not visible on page.
<body>Everything visible on the page goes here.
<title>The tab name in your browser.
<meta charset>Handles special characters. Always include it.
Semantic Tags — Give Your Page Meaning

Semantic tags tell the browser what role each section plays. Professional developers always use these instead of just using generic divs.

<header>The top of the page — logo, site name, navigation.
<nav>Navigation links — menu items.
<main>The primary content of the page.
<section>A distinct section of content e.g. About, Skills.
<article>A self-contained piece of content e.g. a blog post.
<footer>Bottom of the page — copyright, contacts, links.
03
The HTML Skeleton — Study This
This is what every webpage starts with

Read every line. Understand what each tag does before you type it.

index.html
<!DOCTYPE html>
<html lang="en">

  <head>
    <meta charset="UTF-8"/>
    <title>My First Webpage</title>
  </head>

  <body>

    <header>
      <!-- Your name / logo goes here -->
    </header>

    <nav>
      <!-- Navigation links go here -->
    </nav>

    <main>

      <section>
        <!-- About Me section -->
      </section>

      <section>
        <!-- Skills section -->
      </section>

      <section>
        <!-- Hobbies section -->
      </section>

    </main>

    <footer>
      <!-- Copyright / contact info -->
    </footer>

  </body>

</html>
⚠️ Important: This page will look completely blank in the browser right now — and that is CORRECT. You have built the invisible structure. Tomorrow we start filling it with real content.
04
Rules to Follow While Coding
  • ⌨️
    Type every line manually. Do not copy-paste from this guide. Your fingers and brain learn together when you type it yourself.
  • 🔍
    Indent your code. Every tag inside another tag should be pushed in with a Tab. This keeps your code readable — like a professional.
  • ✅
    Always close your tags. If you open <body>, you must close </body>. Missing closing tags cause bugs.
  • 💾
    Save your file as index.html — not index.txt, not webpage.html. The name index.html is the standard.
  • 🌐
    Preview in Chrome. Right-click your file → Open with Chrome. Or use Live Server if you installed it in VS Code.
  • 🚫
    Do not skip ahead. No CSS today. No JavaScript. Just HTML structure. Master the foundation first.

Today's Exercise
Your Task T

Build Your Personal Profile Page Skeleton

Create a new file called index.html. Build a complete HTML skeleton for your own personal profile page. The page should have 4 clearly labelled sections. No styling. No colors. Just clean, correct HTML structure.

  • 1
    Open VS Code (laptop) or Spck/Acode (phone). Create a new file. Save it as index.html
  • 2
    Type the full document structure from memory: <!DOCTYPE html>, <html>, <head>, <title>, <body> — in the correct order.
  • 3
    Inside <body>, add a <header> with your name inside it using an <h1> tag.
  • 4
    Add a <nav> with 3 links inside: About, Skills, Contact. (Links can go nowhere yet — just type them.)
  • 5
    Inside <main>, create 3 <section> blocks — one for About Me, one for Skills, one for Hobbies. Add a short sentence inside each.
  • 6
    Add a <footer> at the bottom with your name and the year — example: "© 2025 Edison"
  • 7
    Open the file in Chrome. It should show plain text — no design. That is correct. Check that all your sections appear.
